Arrays o arreglos


Son un tipo de dato, no son un dato primitivo como los numeros o los strings sino una construccion. Son una lista ordenada
que pueden contener cualquier tipo de elementos, funciones u otros arrays.
Los elementos se guardan dentro de esta variable y son accesibles mediante el uso de un indice.
La forma de crearlos y de llamarlos es la siguente:

var array =[2, 4, 5, 4]

array	// [2, 4, 5, 4]

Si quisiera acceder a un elemento en especifico lo hacemos a travez de los indices.

// array		[2, 	4, 	5, 	4]
// indice		 0	1	2	3

array[2]	//5
